BookStore CRUD Application using Spring and SpringBoot 

Description: 
    This is a BookStore CRUD web application developed using Spring Boot. 
    It allows users to register, login, and perform CRUD operations
    (Create, Read, Update, Delete) on books.

Features:
- User Signup & Login
- Book Management (Add, View, Edit, Delete)
- Form Validation
- Thymeleaf UI
- Spring Boot MVC Architecture

Technologies Used
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- Maven
- HTML & CSS
- MySQL

Project Structure:
src
 ├── main
 │   ├── java
 │   │   └── com.example.BookStore
 │   ├── resources
 │   │   ├── templates
 │   │   ├── static
 │   │   └── application.properties
 └── test

How to run the Project:
1. Clone the repository
   git clone https://github.com/vishnuprasandh61/Crud-Application-Using-Spring-and-Springboot.git
2. Open the project in Eclipse or IntelliJ
3. Configure database in application.properties
4. Run the application
5. Open browser and visit:
   http://localhost:8080
